# Usage
```sh
# Clone this repo:
git clone https://github.com/AlexanderMac/openapi-to-java

# Init your repo:
cd openapi-to-java && rm -rf .git && git init

pnpm install  # install dependencies
pnpm lint     # run linter
pnpm format   # run prettier
pnpm test     # run tests
pnpm coverage # code coverage
```
